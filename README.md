Vamos usar aquele codigo python de soma.

Este código foi desenvolvido em Python com o objetivo de calcular o valor da energia elétrica de uma residência. Primeiro, o programa solicita ao usuário o valor do salário mínimo atual e, em seguida, pergunta a quantidade de quilowatts consumidos na residência.

Depois de receber essas informações, o sistema realiza os cálculos automaticamente. Ele divide o salário mínimo por dois para obter um valor de referência e, posteriormente, multiplica esse resultado pela quantidade de quilowatts informada pelo usuário.

Ao final, o programa exibe na tela o valor total calculado para o pagamento da energia da residência. O código utiliza comandos básicos da linguagem Python, como `input()` para entrada de dados, operações matemáticas e `print()` para mostrar o resultado ao usuário.
